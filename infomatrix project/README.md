# QazaqCompetition (Frontend Only)

MVP веб-проект для регистрации на соревнования в Казахстане.

## Стек
- Frontend: HTML, CSS, Vanilla JS
- Данные: локальное хранилище браузера (`localStorage`)

## Что уже работает
- Регистрация и вход
- Роли: участник, судья, организатор, зритель, админ
- Каталог соревнований + фильтры
- Регистрация участника на соревнование
- Зал славы (награды, опыт, история матчей)
- Live и видео-архив
- Бонусы и история начислений/списаний
- Профиль пользователя (создание и редактирование)
- Судейская панель (оценка + QR-проверка)
- Панель организатора (создание соревнований, услуги, AI-жеребьевка)
- AI-функции: советы, шансы, слабые места, жеребьевка

## Структура
- `script.js` - клиентская логика и локальный API
- `style.css` - единый стиль для всех страниц
- `*.html` - страницы MVP

## Запуск
1. Открой `index.html` в браузере.
2. Все данные хранятся локально в браузере и не требуют сервера.

## Демо-аккаунты
- `participant@qc.local / 123456`
- `judge@qc.local / 123456`
- `organizer@qc.local / 123456`

## Примечание
Если нужно сбросить все локальные данные, очисти `localStorage` в браузере.
